# gulp-cap-helper
Gulp Cap Helper


#Instal
 ``` npm i gulp-cap-helper@git+https://github.com/samundrak/gulp-cap-helper.git ```
 
 
# Simple
  In gulp file
  
  ``` var gulp = require('gulp'); ```
 
  ``` require('gulp-cap-helper')(gulp,[custom task name]); ```
  
  From Terminal
 
  ``` gulp deploy ```
 
  default task name is deploy or provided name
